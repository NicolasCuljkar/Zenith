Multi-utilisateur

Foyer partagé

Deux personnes peuvent partager leurs données au sein d'un foyer. Chacun garde son compte individuel mais accède aux données communes.

🏠Créer ou rejoindre un foyer

  1. Ouvrez les Paramètres (avatar en haut à droite) → section Foyer
  2. Cliquez sur Créer un foyer si vous êtes le premier à configurer
  3. Un code d'invitation (ou QR code) est généré — partagez-le avec l'autre membre
  4. L'autre membre ouvre ses paramètres → Rejoindre avec un code et saisit le code
  5. Les données sont maintenant partagées en temps réel entre les deux comptes

👤Filtrer par membre

Dans chaque onglet, des onglets membres permettent de filtrer les données :

    Fonctionnalité clé

    Ajuster l'épargne et le loisir

    Sous les KPIs, un curseur interactif vous permet de répartir votre reste à vivre entre épargne supplémentaire et loisir, mois par mois.

    Bloc héros — Curseur épargne / loisir
    Reste à vivre · Mai 2026 +420 €
    Revenu net − fixes − variables · prévu
    Épargne 252 € (60 %) Loisir
    252 €
    → épargne ce mois
    168 €
    → loisir ce mois
    1. Le curseur apparaît dans le bloc héros du suivi mensuel, sous les KPIs
    2. Le reste à vivre total est affiché en haut (ex: +420 €)
    3. Glissez le curseur vers la gauche (plus d'épargne) ou la droite (plus de loisir)
    4. Les montants se recalculent en temps réel — Épargne = RAV × % curseur
    5. Ce réglage est sauvegardé pour ce mois spécifiquement et ne modifie pas les autres mois
    Épargne (gauche)

    Plus vous glissez à gauche, plus la part d'épargne augmente. Cette valeur est indicative — elle correspond à ce que vous prévoyez de mettre de côté.

    Loisir (droite)

    La part restante va au budget loisir du mois. Cela influence le calcul des dépenses de loisir autorisées.

    ✓Le % par défaut (60 % épargne / 40 % loisir) est configuré dans vos paramètres de profil. Le curseur du suivi mensuel permet de l'ajuster ponctuellement pour un mois donné.
    !Ce curseur ne déplace pas réellement d'argent. Il sert uniquement à calculer vos prévisions pour le mois. L'épargne réelle est enregistrée séparément dans l'onglet Épargne.

    Corriger une charge ce mois

    Les charges fixes viennent automatiquement du budget prévisionnel. Pour un mois spécifique, vous pouvez corriger leur montant ou les supprimer.

    Charges fixes — Suivi mensuel

    Corriger le montant

    1. Cliquez sur la ligne de la charge fixe dans le suivi mensuel (desktop) ou tapez dessus (mobile)
    2. Un modal s'ouvre avec le montant actuel pré-rempli
    3. Modifiez le montant et ajoutez une note optionnelle (ex: "Régularisation loyer")
    4. Validez — la ligne affiche le badge Corrigé et le montant original barré

    Supprimer une charge ce mois

    1. Sur mobile : swipez la ligne vers la gauche pour révéler le bouton de suppression
    2. Sur desktop : cliquez sur la ligne → bouton "Supprimer ce mois"
    3. La ligne reste visible avec le style Supprimé pour indiquer qu'elle a été exclue
    ✓Les corrections mensuelles ne modifient pas votre budget prévisionnel permanent. Elles ne s'appliquent qu'au mois concerné.

    Ajouter une dépense manuelle (variables, loisir)

    1. Cliquez sur + Ajouter dans la toolbar ou le bouton flottant + (mobile)
    2. Renseignez la désignation, le montant et sélectionnez la catégorie (Variable ou Loisir)
    3. Optionnellement : note et case "Dépense exceptionnelle" (pour exclure des calculs de tendance)
    4. Validez — la dépense s'ajoute dans la section correspondante du mois

    Ajouter & modifier une entrée

    Chaque entrée du budget représente un revenu ou une dépense récurrente. Voici comment les gérer.

    +Créer une nouvelle entrée

    1. Cliquez sur + Ajouter (bouton en haut à droite de la toolbar) ou sur le + dans l'en-tête d'un groupe
    2. Renseignez la désignation (nom de la charge)
    3. Saisissez le montant — le signe est auto-détecté selon la catégorie (les charges sont automatiquement négatives)
    4. Sélectionnez la catégorie : Revenu, Impôt, Fixe, Variable, etc.
    5. Pour les charges Fixes : renseignez le jour de prélèvement (1-31). Ex: 5 = prélevé le 5 de chaque mois.
    6. Choisissez le membre concerné (Nicolas, Carla, ou Commun)
    7. Cliquez sur Enregistrer

    Épargne

    Suivez l'évolution de votre capital épargne mois par mois. Vous saisissez votre solde total en fin de mois — la variation est calculée automatiquement.

    +Enregistrer un mois

    1. Sélectionnez votre membre (onglet en haut)
    2. Cliquez sur le bouton + (flottant sur mobile, toolbar sur desktop)
    3. Choisissez l'année et le mois concerné
    4. Saisissez votre capital épargne total à la fin de ce mois (ex: 6 150 €) — c'est le solde de votre compte épargne, pas ce que vous avez mis de côté ce mois-là
    5. Validez. La variation mensuelle (delta) est calculée automatiquement.
    ℹExemple : vous aviez 5 800 € en avril et 6 150 € fin mai → le delta calculé est +350 € d'épargne pour mai.

    Questions fréquentes

    FAQ

    ?Pourquoi mon RAV ne correspond pas à mon solde bancaire ?

    Le RAV est calculé à partir du budget prévisionnel. Deux raisons possibles :

    • Des charges fixes ont un jour de prélèvement futur — elles ne sont pas encore comptées
    • Vous avez des dépenses variables non encore saisies dans le suivi mensuel

    ?Pourquoi certaines charges fixes sont orange dans le calendrier ?

    Orange = prélèvement à venir dans le mois en cours (date de prélèvement > aujourd'hui). Vert = déjà prélevé. Ces charges oranges sont exclues du RAV réel car elles n'ont pas encore quitté votre compte.

    ?Comment changer le % d'épargne par défaut ?

    Dans Paramètres (votre avatar en haut à droite) → ajustez le paramètre % épargne sur RAV. Ce réglage s'applique à tous les mois par défaut. Vous pouvez l'ajuster ponctuellement via le curseur dans le suivi mensuel.

    ?Le curseur épargne/loisir modifie-t-il mon budget réel ?

    Non. Le curseur dans le suivi mensuel est uniquement prévisionnel pour ce mois. Il calcule combien vous prévoyez d'épargner vs dépenser en loisir. L'épargne réelle doit être enregistrée séparément dans l'onglet Épargne.

    ?Les données sont-elles sauvegardées automatiquement ?

    Oui. Toutes les entrées et corrections sont sauvegardées en base de données dès validation. Les préférences locales (thème, objectifs, % épargne) sont dans votre navigateur et persistent entre sessions.

    ?L'objectif épargne s'applique-t-il aux deux membres ?

    Cela dépend du membre sélectionné. En vue Tous, l'objectif est comparé à l'épargne combinée des deux membres. Filtrez sur votre nom pour comparer uniquement votre épargne personnelle.
